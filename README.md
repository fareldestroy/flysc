
# Farel Destroyer Fly Script

Roblox Fly Script with UI and loading bar.

## Features
- Loading screen with loading bar
- Draggable UI
- Airspeed control (0 - 100)
- Fly toggle
- Close button (hide UI)
- Clean design

## Executor
Works with:
- Delta
- Fluxus
- Synapse style executors

## Usage

Execute:

```lua
loadstring(game:HttpGet("YOUR_RAW_GITHUB_LINK/main.lua"))()
```
